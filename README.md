### Installation
---

1. Clone the repository
2. Open with code editor
3. Split the terminal into two.
    On first run:
    + ` npm i concurrently -g`
    + ` cd client `
    + ` npm install `
    + ` cd .. `
    + ` cd server `
    + ` npm install `
    + ` cd .. `
    + ` npm run dev `
    
    On second terminal run
    + ` cd exercise_tracker `
    + ` pip install -r requirements.txt `
    + ` python app.py `
